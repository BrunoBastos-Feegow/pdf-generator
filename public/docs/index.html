Introduction

API for generating PDFs from HTML

Base URL: http://localhost:8008

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

PDF generation

Basic PDF generation

Generate PDFs from HTML

Example request:
curl --request POST \
    "http://localhost:8008/api/basic" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"html\": \"<html><body><h1>Teste<\\/h1><\\/body><\\/html>\"
}"

const url = new URL(
    "http://localhost:8008/api/basic"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "html": "<html><body><h1>Teste<\/h1><\/body><\/html>"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, Success):


{
"Content-Type": "application/pdf",
"Content-Disposition": "inline; filename="2021-09-28 14:09:00.pdf"",
"Cache-Control": "no-cache, no-store, max-age=0, must-revalidate"
}

Example response (422, Validation error):


{
    "message": "The given data was invalid.",
    "errors": {
        "html": [
            "O campo html é obrigatório."
        ]
    }
}

Example response (500, Internal server error):


{
    "message": "Internal server error"
}
