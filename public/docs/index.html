Introduction

API for generating PDFs from HTML

Base URL: http://localhost:8008

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

PDF generation

Basic PDF generation

Generate PDFs from HTML

Example request:
curl --request POST \
    "http://localhost:8008/api/basic" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"html\": \"<html><body><h1>Teste<\\/h1><\\/body><\\/html>\"
}"

const url = new URL(
    "http://localhost:8008/api/basic"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "html": "<html><body><h1>Teste<\/h1><\/body><\/html>"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, Success):


{
"Content-Type": "application/pdf",
"Content-Disposition": "inline; filename="2021-09-28 14:09:00.pdf"",
"Cache-Control": "no-cache, no-store, max-age=0, must-revalidate"
}

Example response (422, Validation error):


{
    "message": "The given data was invalid.",
    "errors": {
        "html": [
            "O campo html é obrigatório."
        ]
    }
}

Example response (500, Internal server error):


{
    "message": "Internal server error"
}

Advanced PDF generation

Generate PDFs from HTML with header and footer repeating them in every page

Example request:
curl --request POST \
    "http://localhost:8008/api/advanced" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"settings\": {
        \"useHeader\": true,
        \"useFooter\": true,
        \"paperSize\": \"A4\",
        \"orientation\": \"portrait\"
    },
    \"header\": \"<html><body><h1>Header<\\/h1><\\/body><\\/html>\",
    \"body\": \"<html><body><h1>Body<\\/h1><\\/body><\\/html>\",
    \"footer\": \"<html><body><h1>Footer<\\/h1><\\/body><\\/html>\"
}"

const url = new URL(
    "http://localhost:8008/api/advanced"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "settings": {
        "useHeader": true,
        "useFooter": true,
        "paperSize": "A4",
        "orientation": "portrait"
    },
    "header": "<html><body><h1>Header<\/h1><\/body><\/html>",
    "body": "<html><body><h1>Body<\/h1><\/body><\/html>",
    "footer": "<html><body><h1>Footer<\/h1><\/body><\/html>"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, Success):


{
"Content-Type": "application/pdf",
"Content-Disposition": "inline; filename="2021-09-28 14:09:00.pdf"",
"Cache-Control": "no-cache, no-store, max-age=0, must-revalidate"
}

Example response (422, Validation error):


{
    "message": "The given data was invalid.",
    "errors": {
        "settings": [
            "O campo settings é obrigatório."
        ],
        "header": [
            "O campo header é obrigatório."
        ],
        "body": [
            "O campo body é obrigatório."
        ],
        "footer": [
            "O campo footer é obrigatório."
        ]
    }
}

Example response (500, Internal server error):


{
    "message": "Internal server error"
}

Request      

POST api/advanced

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

settings   object   

The settings for the PDF generation.
useHeader   boolean  optional  

Whether to use header or not. Example: true

useFooter   boolean  optional  

Whether to use footer or not. Example: true

paperSize   string  optional  

The paper size. Example: A4

orientation   string  optional  

The orientation. Example: portrait

header   string   

The HTML for the header. Example: <html><body><h1>Header</h1></body></html>

body   string   

The HTML for the body. Example: <html><body><h1>Body</h1></body></html>

footer   string   

The HTML for the footer. Example: <html><body><h1>Footer</h1></body></html>
